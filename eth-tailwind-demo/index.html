How It Works

Implementing ETH Zürich design tokens in your Tailwind CSS project is straightforward:

  1. Download the ETH Tailwind configuration file
  2. Import it into your Tailwind config
  3. Start using ETH-specific class names like bg-eth-blue instead of bg-blue-500

The ETH Tailwind Configuration

Here's a snippet of the ETH Tailwind config file:

  theme: {
    extend: {
