Design-Tokens für Entwickler

Was ist der Design-Token Ansatz?

Design-Tokens sind benannte Werte, die die grundlegenden visuellen Designattribute der ETH Zürich definieren. Sie bilden die Basis für ein konsistentes Erscheinungsbild über alle digitalen Produkte hinweg. Mit diesem Ansatz stellen wir die Design-Grundlagen in einer maschinenlesbaren Form bereit, die einfach in verschiedene Entwicklungsumgebungen integriert werden kann.

Hauptmerkmale

  • Maschinenlesbare Werte - Alle Design-Werte in JSON-Format mit automatischen Exporten für verschiedene Plattformen
  • Mehrere Ausgabeformate - CSS-Variablen, SCSS, Tailwind-Konfiguration je nach Bedarf
  • Einfache Integration - Direkter Import in Ihre bestehenden Projekte ohne Framework-Abhängigkeit
  • Zentrale Design-Wahrheit - Alle Werte an einem Ort für einfache Aktualisierungen

ETH Design System im Figma Dev Mode

Das ETH Design System ist in Figma verfügbar, wo Sie alle aktuellen Design-Tokens und Komponenten einsehen können. Mit Figma Dev Mode können Designer und Entwickler nahtlos zusammenarbeiten.

Der Figma Dev Mode bietet folgende Vorteile für Entwickler:

  • Direkte Inspektion von Design-Komponenten und -Tokens
  • Exakte CSS-Werte für jedes Element
  • Live-Updates, wenn sich das Design ändert
  • Möglichkeit, spezifische Implementierungsdetails zu kommentieren

Implementation

Die Design-Tokens können einfach in Ihre Projekte eingebunden werden. Hier sind Beispiele für verschiedene Formate:

JSON (Standard-Format)

JSON 
{
  "color": {
    "brand": {
      "blue": { "value": "#215CAF" },
      "blue-80": { "value": "#4D7DBF" },
      "blue-60": { "value": "#7A9DCF" },
      "blue-40": { "value": "#A6BEDF" },
      "blue-20": { "value": "#D3DEEF" }
    },
    "semantic": {
      "success": { "value": "#2E7B32" },
      "warning": { "value": "#FDA726" },
      "error": { "value": "#E2001A" },
      "info": { "value": "#215CAF" }
    }
  },
  "spacing": {
    "2": { "value": "0.5rem" },
    "4": { "value": "1rem" },
    "6": { "value": "1.5rem" },
    "8": { "value": "2rem" },
    "12": { "value": "3rem" }
  },
  "fontFamily": {
    "primary": { "value": "\"DIN\", Helvetica, Arial, sans-serif" }
  },
  "borderRadius": {
    "none": { "value": "0" },
    "sm": { "value": "2px" },
    "md": { "value": "4px" },
    "lg": { "value": "8px" }
  }
}

CSS-Variablen

CSS 
/* Einbinden der CSS-Variablen */
@import url('path/to/eth-tokens.css');

/* Verwendung in Ihrem CSS */
.button {
  background-color: var(--eth-blue);
  padding: var(--spacing-2) var(--spacing-4);
  border-radius: var(--border-radius-md);
  color: var(--white);
}

SCSS-Variablen

SCSS 
// Einbinden der SCSS-Variablen
@import 'path/to/eth-tokens.scss';

// Verwendung in Ihrem SCSS
.button {
  background-color: $eth-blue;
  padding: $spacing-2 $spacing-4;
  border-radius: $border-radius-md;
  color: $white;
}

// tailwind.config.js

  ],

Typografie

Schriftfamilien, -größen und -gewichte für konsistente Textdarstellung.

  • font-size-lg: 1.125rem (18px)

Integration mit Entwicklungs-Workflows

Es gibt mehrere Möglichkeiten, die ETH Design-Tokens in Ihre Entwicklungs-Workflows zu integrieren:

  1. Direkter JSON-Import: Laden Sie die JSON-Tokens herunter und integrieren Sie sie mit Ihrem Build-Prozess
  2. CSS-Variablen: Binden Sie die vorgenerierten CSS-Variablen in Ihr Projekt ein
  3. Figma Dev Mode: Greifen Sie direkt über die Figma API auf die aktuellsten Token-Werte zu
  4. Token-Transformation: Verwenden Sie Tools wie Style Dictionary, um die Tokens in Ihr bevorzugtes Format zu konvertieren

Ressourcenschonender Ansatz

Der Design-Token-Ansatz bietet mehrere Vorteile für Teams mit begrenzten Ressourcen:

  • Geringe Wartung: Tokens müssen nur aktualisiert werden, wenn sich das Markendesign ändert
  • Framework-unabhängig: Keine Bindung an spezifische Frontend-Frameworks
  • Skalierbar: Beginnen Sie mit Grundtokens und erweitern Sie bei Bedarf
  • Automatisierbar: Einfache Automatisierung der Token-Erstellung und -Aktualisierung mit AI-Unterstützung
  • Direkte Designer-Entwickler-Zusammenarbeit: Figma Dev Mode ermöglicht effiziente Kommunikation
