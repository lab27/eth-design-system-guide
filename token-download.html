What are Design Tokens?

Design tokens are the visual design atoms of the design system — specifically, they are named entities that store visual design attributes. We use them in place of hard-coded values to ensure flexibility and maintain a consistent visual language.

Consistency

Tokens ensure all ETH Zürich digital products share the same visual foundation.

Efficiency

Implement designs faster using predefined tokens rather than creating styles from scratch.

Accessibility

Our tokens are created with accessibility guidelines in mind.

Scalability

Easily adapt to different platforms and screen sizes with responsive token values.

Token Categories

Colors

Color tokens include primary ETH Zürich blues, secondary colors, neutral grays, and semantic colors for statuses.

Typography

Typography tokens include font families, sizes, weights, and line heights used across ETH Zürich properties.

Spacing

Spacing tokens define consistent margins, padding, and layout spacing throughout the design system.

Other Tokens

Additional tokens include shadows, border radiuses, breakpoints, and more.
