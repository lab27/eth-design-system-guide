Tailwind Preset & UI-Muster für ETH Entwickler

Der Tailwind-Ansatz

Statt einer komplett eigenen Lösung nutzen wir das beliebte Tailwind CSS-Framework und erweitern es mit ETH-spezifischen Design-Tokens. Dieser Ansatz bietet die perfekte Balance zwischen Flexibilität und Konsistenz: Entwickler können die vertraute Tailwind-Syntax nutzen, während die ETH Zürich Markenidentität gewahrt bleibt.

Schnellstart

In drei einfachen Schritten können Sie das ETH Tailwind-Preset in Ihr Projekt integrieren:

1. Installation

Bash 
npm install tailwindcss @ethzurich/tailwind-preset

2. Tailwind konfigurieren

JavaScript 
// tailwind.config.js
module.exports = {
  presets: [
    require('@ethzurich/tailwind-preset')
  ],
  content: [
    './src/**/*.{html,js,jsx,ts,tsx,vue}'
  ]
}

3. CSS einbinden

CSS 
/* main.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

Vorteile des Tailwind-Ansatzes

  • Geringer Wartungsaufwand: Das Preset wird automatisch mit neuen Tailwind-Versionen aktualisiert
  • Vertraute Syntax: Entwickler können bestehende Tailwind-Kenntnisse nutzen
  • JIT-Kompilierung: Nur die tatsächlich verwendeten Klassen werden in das finale CSS aufgenommen
  • Responsive Design: Vollständige Unterstützung für responsive Breakpoints
  • Erweiterbar: Einfach um eigene Komponenten und Plugins erweiterbar
